Что такое структуры данных?
Программная единица, позволяющая хранить и обрабатывать множество однотипных и/или логически связанных данных в вычислительной технике

Зачем они нужны?
Программисты в своей профессиональной деятельности широко используют структуры данных для удобного хранения информации

Что такое Hash Map?
Это структура данных, реализующая интерфейс ассоциативного массива, а именно, она позволяет хранить пары (ключ, значение) и выполнять три операции: операцию добавления новой пары, операцию поиска и операцию удаления пары по ключу

Что такое Linked List?
Связный список - это структура данных, в которой несколько значений хранятся линейно. Каждое значение содержит своё собственное значение узла, а также содержит данные вместе со ссылкой на следующий узел в списке

В каких случаях использовать Linked List и Hash Map?
С помощью связных списков реализуются файловые системы, хеш-таблицы и списки смежности. С помощью хэш мэп в основном реаилизется хранение данных по принципу "ключ: значение"
