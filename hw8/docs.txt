Какую функцию выполняет запрос SQL CREATE?
Создание новой таблицы, базы данных

Какую функцию выполняет запрос SQL INSERT?
Вставка новой записи в таблицу

Какую функцию выполняет запрос SQL SELECT?
Выборка данных из таблиц(ы). Возможна выборка по условию

Какую функцию выполняет запрос SQL UPDATE?
Обновление данных в таблицах

Какую функцию выполняет запрос SQL DROP?
Удаляет все таблицы из базы данных и саму базу данных
